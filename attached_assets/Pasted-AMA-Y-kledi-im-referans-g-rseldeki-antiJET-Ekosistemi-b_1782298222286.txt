AMAÇ

Yüklediğim referans görseldeki “ŞantiJET Ekosistemi” bölümünü web sitesinde birebir üret.

Bu bir yeniden tasarım çalışması değildir.

Görseldeki kompozisyon, renkler, kart yerleşimleri, ikonlar, glow efektleri, daire yapısı ve genel estetik korunacaktır.

Sadece web ortamına uygun motion (animasyon) ve etkileşim eklenecektir.

KRİTİK KURAL

* Görseldeki tasarımı değiştirme.
* Kartları yeniden konumlandırma.
* Yeni kart ekleme.
* Renk değiştirme.
* Yazı fontunu değiştirme.
* Layout’u modernleştirmeye çalışma.
* UI/UX iyileştirmesi yapma.

Amaç görseli web ortamında canlı hale getirmektir.

KULLANILACAK TEKNOLOJİ

* React
* TypeScript
* Tailwind CSS
* Framer Motion
* SVG Animasyonları
* Responsive yapı

Animasyonlar GPU dostu olmalı.

60 FPS hedeflenmeli.

VERİ AKIŞI ANİMASYONU

Kartlar ile merkez arasındaki mavi bağlantılar korunacak.

Bu çizgiler üzerinde sürekli veri akışı olacak.

SVG path kullan.

Path boyunca:

* mavi ışık noktaları
* enerji parçacıkları
* veri paketleri

hareket etsin.

Yön:

Kart → Merkez
Merkez → Kart

sürekli çift yönlü akış.

Hız:

3-5 saniye arası.

Kesintisiz döngü.

MAVİ IŞIK HATTI
